## 📱 Overview

**Amigos** is a feature-rich social media application built with cutting-edge web technologies. Experience seamless content sharing, real-time interactions, and beautiful UI animations.

🔗 **Live Demo:** https://social-app-test-coral.vercel.app/

---

## ✨ Features

### 🔐 Authentication
- User registration with validation
- Secure login/logout
- Dark mode preference persistence

### 📝 Posts
- Create posts with images (URL)
- Edit/delete your own posts
- Like/unlike with animated feedback
- Comment on posts
- Share posts (copy link)

### 👥 Social
- Follow/unfollow users
- View follower/following counts
- Search for users by name/username
- Suggested users based on activity

### 🎨 UI/UX
- Fully responsive (mobile, tablet, desktop)
- Glassmorphism design effects
- Smooth page transitions
- Skeleton loading states
- Micro-interactions and animations

### 🚀 Performance
- Vite for fast builds and HMR
- Debounced search
- Optimized re-renders with Context API
- Lazy loading ready architecture
